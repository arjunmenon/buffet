Buffet

Buffet is website code that you can use to show stuff in a scroller.

← →

Javascript

$(document).ready(function () {
  $('#scroll').buffet({
    scroll_by : 2,
    speed     : 400,
    easing    : 'easeOutExpo',
    next      : $('#next'),
    prev      : $('#prev')
  });
});

HTML

<div id="scroll_mask">
